# Omar Abbas | Developer Portfolio

A premium, high-performance developer portfolio built with **Astro**, **React**, **Tailwind CSS**, and **Shadcn UI**. This portfolio features a "techno-functional" approach, showcasing both software engineering expertise and executive management skills.

## 🚀 Key Features

- **Dynamic Persona Toggle**: Switch between "Engineer" and "Executive" views to highlight different career strengths.
- **Content-Driven Architecture**: Managed via Astro Content Collections and Markdown.
- **Premium UX/UI**: Sleek dark mode, responsive design, and smooth micro-animations.
- **Interactive Resume**: Filterable experience timeline and comprehensive skills breakdown.
- **Production Ready**: Optimized for speed and deployed via Netlify.

## 🛠️ Tech Stack

- **Framework**: [Astro](https://astro.build/)
- **UI Components**: [Shadcn UI](https://ui.shadcn.com/) & [React](https://reactjs.org/)
- **Styling**: [Tailwind CSS](https://tailwindcss.com/)
- **Deployment**: [Netlify](https://www.netlify.com/)
- **Content**: Markdown & Zod for schema validation

## 🧞 Commands

All commands are run from the root of the project:

| Command                   | Action                                           |
| :------------------------ | :----------------------------------------------- |
| `npm install`             | Installs dependencies                            |
| `npm run dev`             | Starts local dev server at `localhost:4321`      |
| `npm run build`           | Build your production site to `./dist/`          |
| `npm run preview`         | Preview your build locally, before deploying     |
| `npm run astro check`     | Run diagnostics on your codebase                 |
